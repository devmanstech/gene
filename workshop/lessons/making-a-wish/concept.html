When a wish is made, a few things happen. First genie checks that the wish is in context. If it is, then the wish's action is invoked. If a magicWord is provided, then genie internally associates that magic word with that wish for better sorting in the future. This is how genie learns the user as time goes on (more on this later).
Genie will also increment the wish's data.timesMade.total and data.timesMade.magicWords[magicWord] values.
