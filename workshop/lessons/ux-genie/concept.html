ux-genie is an AngularJS module for using genie in an AngularJS context. If you're not familiar with AngularJS, just enjoy seeing how easy it is to do stuff, and then go learn AngularJS :-)

Module Directives

ux-lamp

This is the magical thing you see when you hit the ctrl + space keys on this page. The lamp is the user's interface with genie.

genie-wish

By putting this directive on an element, it will allow you to add a genie wish to trigger an event on that element.

To use:

Make sure you include both genie.js and uxGenie.js on your page, then load the 'uxGenie' module as a dependency in yours.
